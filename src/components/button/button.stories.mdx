# &lt;sui-button&gt;

A component used for enabling user click interactions. Can be used as a classic "button" (`<button>`) or as a linked button (`<a>`).

## Features:

- Shadow DOM
- Defaults to `<button>`
- Redefine element using `as` prop

<sui-button bg="gray" p="2" href="#">Source code on Github</sui-button>
