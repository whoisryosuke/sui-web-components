# &lt;sui-base&gt;

A component used for passing theme values (or CSS custom properties) to nested components. Contains all "global" design tokens and theme logic, like colors and typography settings.

Not required if you use your own CSS with any necessary custom properties (colors, spacing, etc). Required for using themed utility props, such as the `color` or `margin` utility props.

## Features:

- Shadow DOM
- Doesn't create a DOM element, only returns slotted content
