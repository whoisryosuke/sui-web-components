# &lt;sui-switch&gt;

A component used for capturing user input through a boolean toggle.

## Features:

- Uses internal method to set `checked` prop
- Uses events to track changes to `checked` prop
- [WAI-ARIA compliant](https://www.w3.org/TR/wai-aria-practices-1.1/#checkbox)
